# hello-KLDivLoss-and-CrossEntropyLoss

Learn PyTorch `LogSoftmax`, `KLDivLoss`, and `CrossEntropyLoss` with runnable examples.

```bash
uv run python learn_log_softmax.py
uv run python learn_kl_div_loss.py
uv run python learn_cross_entropy_loss.py
```

Each script prints all input subcases and documents the captured output in inline comments next to the code.
